# Trilha de Aprendizado em Java

## 1. Variáveis, Tipos de Dados e Operadores:

Nesta seção, combinaremos exercícios relacionados a declaração de **variáveis**, tipos de dados e operadores, cobrindo os blocos de construção fundamentais da programação Java.

1. **Declarações e Atribuições de Variáveis**: Declare variáveis de vários tipos de dados (`int`, `double`, `char`, `boolean`) e atribua valores a elas. Imprima seus valores no console para verificar.
2. **Conversão de Temperatura**: Escreva um programa para converter valores de temperatura entre Celsius e Fahrenheit usando operações aritméticas e entrada do usuário.
3. **Cálculos de Retângulo**: Desenvolva um programa para calcular a área e o perímetro de um retângulo. Solicite ao usuário que insira o comprimento e a largura, execute os cálculos e exiba os resultados.
4. **Troca de Variáveis**: Escreva código para trocar os valores de duas variáveis sem usar uma terceira variável temporária. Implemente isso usando operações aritméticas.
5. **Determinação Par/Ímpar**: Crie um programa que verifica se um determinado número inteiro é par ou ímpar. Utilize o operador módulo (`%`) para determinar a divisibilidade por 2.
6. **Exploração da Classe Math**: Explore os recursos da classe `Math` escrevendo código para calcular a raiz quadrada, o valor absoluto e a potência de um número. Use os métodos apropriados da classe `Math` para isso.
7. **Geração de Números Aleatórios**: Gere um número aleatório dentro de um intervalo especificado (por exemplo, entre 1 e 100) usando o método `Math.random()`.
8. **Cálculos de Círculo**: Desenvolva um programa para calcular a área e a circunferência de um círculo. Utilize fórmulas matemáticas e constantes (por exemplo, `PI`) da classe `Math` para realizar esses cálculos.

## 2. Estruturas Condicionais:

Esta seção se concentrará no uso de instruções condicionais (`if-else`, `switch`) para controlar o fluxo de execução com base em certas condições.

1. **Avaliação de Notas do Aluno**: Escreva um programa para determinar se um aluno foi aprovado ou reprovado com base na nota do exame. Use uma instrução `if-else` para verificar se a nota está acima ou abaixo do limite de aprovação.
2. **Encontrando o Maior Número**: Crie um programa para encontrar o maior entre três números fornecidos. Empregue instruções `if-else` aninhadas para comparar os números e identificar o valor máximo.
3. **Atribuição de Notas**: Desenvolva um programa que atribui notas de letras (`A`, `B`, `C`, etc.) aos alunos com base em suas pontuações percentuais. Use instruções `if-else` ou `switch` para categorizar as pontuações em diferentes faixas de notas.
4. **Dia da Semana**: Implemente um programa que exibe o dia da semana correspondente a um determinado número (1-7) usando uma instrução `switch`. Lide com valores de entrada inválidos adequadamente.
5. **Calculadora Simples**: Projete um aplicativo de calculadora básico que realiza operações aritméticas (`+`, `-`, `*`, `/`) em dois números com base na entrada e seleção do usuário. Use instruções `if-else` ou `switch` para executar a operação escolhida.

## 3. Estruturas de Repetição:

Esta seção abordará o uso de loops (`for`, `while`) para repetir blocos de código por um certo número de iterações ou até que uma condição específica seja atendida.

1. **Impressão de Números**: Escreva um programa para imprimir números de 1 a 10 (ou qualquer intervalo fornecido) usando um loop `for`.
2. **Cálculo Fatorial**: Crie um programa para calcular o fatorial de um inteiro não negativo usando um loop `while`.
3. **Sequência de Fibonacci**: Gere a sequência de Fibonacci até um certo limite usando um loop `for` ou `while`. A sequência de Fibonacci começa com 0 e 1, e cada número subsequente é a soma dos dois anteriores.
4. **Verificação de Número Primo**: Desenvolva um programa que verifica se um determinado número é primo. Um número primo é um número natural maior que 1 que não possui divisores positivos além de 1 e ele mesmo.
5. **Tabuada**: Gere a tabuada para um determinado número usando loops aninhados. A tabela deve exibir o produto do número e cada inteiro de 1 a 10.

## 4. Funções e Métodos:

Esta seção apresentará funções e métodos, permitindo que você crie blocos de código reutilizáveis e modularize seus programas.

1. **Função Soma**: Escreva uma função chamada `soma` que recebe dois argumentos inteiros e retorna sua soma. Demonstre o uso da função em seu programa principal.
2. **Função Par/Ímpar**: Crie uma função chamada `ePar` que recebe um argumento inteiro e retorna verdadeiro se o número for par e falso caso contrário.
3. **Inversão de String**: Escreva uma função chamada `inverterString` que recebe um argumento de string e retorna a versão invertida da string. Você pode conseguir isso iterando pela string e construindo uma nova string invertida.
4. **Pesquisa em Array**: Implemente uma função chamada `encontrar` que recebe um array de inteiros e um valor alvo como argumentos. A função deve procurar o valor alvo dentro do array e retornar seu índice se encontrado, ou -1 se o valor alvo não estiver presente.
5. **Fatorial Recursivo**: Implemente uma função recursiva chamada `fatorial` que calcula o fatorial de um inteiro não negativo. A função deve se chamar com um valor menor até atingir o caso base de `0!` (que é 1).

## 5. Arrays e ArrayLists:

Esta seção explorará o uso de arrays e ArrayLists, que são estruturas de dados para armazenar coleções de elementos.

1. **Declaração e Impressão de Array**: Declare e inicialize um array de inteiros com alguns valores. Escreva código para imprimir todos os elementos do array usando um loop.
2. **Encontrando Extremos do Array**: Desenvolva um programa para encontrar os valores máximo e mínimo dentro de um array de inteiros. Itere pelo array, mantendo o controle dos valores mais alto e mais baixo encontrados.
3. **Classificação de Array**: Implemente um ou mais algoritmos de classificação (por exemplo, classificação bolha, classificação por inserção, classificação por seleção) para classificar um array de inteiros em ordem crescente.
4. **Criação e Manipulação de ArrayList**: Crie um ArrayList para armazenar strings. Adicione várias strings ao ArrayList usando o método `adicionar()`. Experimente outros métodos ArrayList, como `remover()`, `obter()` e `tamanho()`.
5. **Pesquisa e Remoção de ArrayList**: Escreva código para procurar uma string específica dentro de um ArrayList e removê-la se encontrada. Use os métodos `indiceDe()` e `remover()` para fazer isso.

## 6. Strings em Java:

Esta seção se aprofundará no trabalho com strings, que são sequências de caracteres que representam texto em Java.

1. **Comparação de Strings**: Escreva um programa para comparar duas strings e determinar se elas são iguais usando o método `equals()`. Explore outros métodos de comparação como `equalsIgnoreCase()` para comparações que não diferenciam maiúsculas de minúsculas.
2. **Comprimento da String**: Encontre o comprimento de uma determinada string usando o método `length()`.
3. **Conversão de Caixa**: Implemente código para converter uma string para maiúsculas usando o método `toUpperCase()` e para minúsculas usando o método `toLowerCase()`.
4. **Extração de Substring**: Extraia uma parte específica (substring) de uma string usando o método `substring()`. Experimente diferentes índices iniciais e finais para extrair várias substrings.
5. **Contagem de Caracteres**: Escreva um programa para contar as ocorrências de um caractere específico dentro de uma string. Itere pela string e incremente um contador cada vez que o caractere desejado for encontrado.

## 7. Classes e Objetos:

Esta seção apresentará conceitos de programação orientada a objetos (POO), incluindo classes, objetos, atributos e métodos.

1. **Classe Aluno**: Crie uma classe `Aluno` com atributos como nome, idade e `numeroMatricula`. Defina um construtor para inicializar esses atributos e fornecer métodos getter e setter para acessá-los e modificá-los.
2. **Classe Retângulo**: Projete uma classe `Retangulo` com atributos `comprimento` e `largura`. Implemente métodos para calcular a área e o perímetro do retângulo.
3. **Sistema de Conta Bancária**: Desenvolva um sistema básico de conta bancária usando classes e objetos. Crie uma classe `ContaBancaria` com atributos como `numeroConta` e `saldo`. Implemente métodos para depositar e sacar dinheiro e garanta que o saldo não fique negativo.

## 8. Herança e Polimorfismo:

Esta seção explorará herança e polimorfismo, que são conceitos-chave de POO para criar relacionamentos hierárquicos entre classes e alcançar a reutilização de código.

1. **Hierarquia de Formas**: Crie uma classe base chamada `Forma` com métodos para calcular a área e o perímetro. Derive subclasses como `Circulo`, `Retangulo` e `Triangulo` da classe `Forma` e sobrescreva os métodos de área e perímetro para fornecer implementações específicas para cada forma.
2. **Uso Polimórfico de Formas**: Demonstre o polimorfismo criando objetos de diferentes formas (círculo, retângulo, triângulo) e chamando seus métodos de área e perímetro por meio de uma referência do tipo de classe base (`Forma`). Observe como o método apropriado é invocado com base no tipo de objeto real em tempo de execução.
3. **Hierarquia da Empresa**: Implemente uma hierarquia de classes que represente uma estrutura de empresa com funcionários e gerentes. A classe `Funcionario` pode ter atributos como nome, id e salario. A classe `Gerente` pode herdar de `Funcionario` e ter atributos adicionais como departamento e métodos para gerenciar funcionários.
4. **Classes Abstratas e Interfaces**: Explore o conceito de classes abstratas e interfaces. Crie uma classe abstrata `Animal` com um método abstrato `emitirSom()`. Defina subclasses concretas como `Cachorro`, `Gato` e `Passaro` que herdam de `Animal` e fornecem suas próprias implementações do método `emitirSom()`.
5. **Exemplo de Interface**: Projete uma interface chamada `CalculadoraDeForma` com métodos para calcular área e perímetro. Implemente essa interface em várias classes de formas (por exemplo, `Circulo`, `Retangulo`). Isso demonstra como as interfaces podem definir contratos para as classes seguirem, promovendo flexibilidade e desacoplamento do código.

## 9. Estruturas de Dados Avançadas: Pilhas, Filas, Árvores e Tabelas Hash:

Esta seção abordará estruturas de dados avançadas que fornecem maneiras eficientes de organizar e acessar dados para várias tarefas computacionais.

1. **Implementação de Pilha**: Implemente uma estrutura de dados de pilha usando um array ou ArrayList. A pilha deve suportar operações como empilhar (para adicionar um elemento ao topo), desempilhar (para remover o elemento do topo), espiar (para visualizar o elemento do topo sem removê-lo) e estaVazia (para verificar se a pilha está vazia).
2. **Implementação de Fila**: Implemente uma estrutura de dados de fila usando uma lista encadeada ou um array. A fila deve suportar operações como enfileirar (para adicionar um elemento à parte traseira), desenfileirar (para remover um elemento da frente), espiar (para visualizar o elemento da frente sem removê-lo) e estaVazia (para verificar se a fila está vazia).
3. **Simulação de Fila**: Simule um sistema de filas do mundo real, como uma fila de banco ou uma fila de impressora, usando sua implementação de fila.
4. **Árvore de Busca Binária**: Crie uma estrutura de dados de árvore de busca binária e implemente operações como inserção, exclusão e pesquisa. Uma árvore de busca binária mantém uma ordem específica (filho esquerdo menor que pai, filho direito maior que pai), o que permite pesquisa e classificação eficientes.
5. **Exploração de Árvores**: Explore diferentes tipos de árvores, como árvores AVL (árvores de busca binária autobalanceadas) e árvores B (usadas em bancos de dados e sistemas de arquivos). Entenda suas características e casos de uso.
6. **Implementação de Tabela Hash**: Implemente uma estrutura de dados de tabela hash e experimente diferentes funções hash e técnicas de resolução de colisão (por exemplo, encadeamento separado, endereçamento aberto). Tabelas hash fornecem armazenamento e recuperação eficientes de chave-valor.

## 10. Algoritmos de Busca e Classificação:

Esta seção abordará vários algoritmos para pesquisar e classificar dados de forma eficiente.

1. **Busca Linear**: Implemente o algoritmo de busca linear, que itera por um array sequencialmente para encontrar um valor alvo. Analise sua complexidade de tempo, que geralmente é O(n), onde n é o número de elementos no array.
2. **Busca Binária**: Implemente o algoritmo de busca binária, que funciona de forma eficiente em arrays classificados dividindo repetidamente o intervalo de busca pela metade. Analise sua complexidade de tempo, que é O(log n), tornando-o muito mais rápido do que a busca linear para grandes conjuntos de dados.
3. **Implementações de Algoritmos de Classificação**: Implemente vários algoritmos de classificação, incluindo classificação bolha, classificação por inserção, classificação por seleção, classificação por mesclagem e classificação rápida. Entenda como cada algoritmo funciona e analise sua complexidade de tempo e espaço.
4. **Comparação de Algoritmos**: Compare o desempenho de diferentes algoritmos de classificação com base em fatores como o número de elementos, a ordem inicial dos dados e a memória disponível.
5. **Seleção de Algoritmos**: Escolha o algoritmo de classificação apropriado com base nos requisitos específicos de sua aplicação, considerando fatores como o tamanho e a natureza dos dados e o equilíbrio desejado entre eficiência de tempo e espaço.

## Conclusão:

Lembre-se de que este caminho de aprendizagem fornece uma abordagem estruturada para construir uma base sólida na programação Java. Ao se envolver ativamente com os exercícios em cada seção, experimentar diferentes abordagens e buscar recursos de aprendizagem adicionais, você pode desenvolver efetivamente suas habilidades de programação Java e embarcar na construção de projetos mais complexos e interessantes.

## Dicas Adicionais:

- **Pratique Regularmente**: A prática consistente é fundamental para solidificar sua compreensão dos conceitos Java e melhorar suas habilidades de codificação. Reserve um tempo dedicado para exercícios e projetos de codificação.
- **Experimente e Explore**: Não tenha medo de experimentar diferentes abordagens para a resolução de problemas. Explore várias bibliotecas, frameworks e ferramentas disponíveis no ecossistema Java.
- **Busque Ajuda e Colaboração**: Junte-se a comunidades online, fóruns ou grupos de estudo para se conectar com outros alunos de Java e desenvolvedores experientes. Faça perguntas, compartilhe seu conhecimento e colabore em projetos.
- **Construa Projetos**: Aplique seu aprendizado a projetos do mundo real. Comece com projetos pequenos e gerenciáveis e aumente gradualmente a complexidade à medida que ganha confiança. Construir projetos reforçará sua compreensão e demonstrará suas habilidades.
- **Mantenha-se Atualizado**: A linguagem de programação Java e seu ecossistema estão em constante evolução. Mantenha-se atualizado com as últimas tendências, bibliotecas e práticas recomendadas por meio de blogs, artigos e recursos online.

Lembre-se de que a jornada de aprender Java é contínua e recompensadora. Abrace os desafios, continue praticando e aproveite o processo de se tornar um programador Java proficiente!
